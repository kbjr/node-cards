Module: rand

Provides access to the randomization tools and config used for shuffling cards.

const { rand } = require('cards');

The optional ARC4 PRNG can be enabled for randomization if you want to be able to provide a seed value for the PRNG. Otherwise, the built-in Math.random is used.

const { rand } = require('cards');

// Enable the ARC4 PRNG and provide a seed value
rand.useArc4(true);
rand.seedArc4('some seed value');
